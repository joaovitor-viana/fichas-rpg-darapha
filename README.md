# Fichas de RPG da Rapha

Um sistema completo de gerenciamento de fichas de RPG com estilo sombrio e gótico.

## Funcionalidades
- **Autenticação**: Login seguro via Supabase.
- **Ficha do Jogador**: Gerenciamento de Vida, Fome, Sede, Atributos e Inventário.
- **Upload de Tokens**: Armazenamento de imagens de personagens no Supabase Storage.
- **Visão do Mestre**: Listagem em tempo real de todos os jogadores ativos na campanha.
- **Design Sombrio**: Interface personalizada com temas de pergaminho, ferro e teias de aranha.

## Tecnologias
- **Frontend**: React, Vite, Tailwind CSS.
- **Backend**: Supabase (Auth, DB, Storage).
- **Deployment**: Vercel.

## Como usar
1. Crie uma conta ou use as contas de teste.
2. Acesse sua ficha para atualizar seus atributos.
3. Clique em 'SELAR' para salvar as alterações no grimório.
